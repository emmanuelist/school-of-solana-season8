# Solana Counter dApp Frontend

A decentralized counter application built on Solana using Next.js and Anchor.

## Features

- Connect Solana wallets (Phantom, Solflare)
- Initialize personal counter accounts
- Increment and reset counters
- View total increments history

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Solana Web3.js
- Anchor Framework
- Wallet Adapter

## Setup

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Run development server:
\`\`\`bash
npm run dev
\`\`\`

3. Open [http://localhost:3000](http://localhost:3000)
